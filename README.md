<h2>서버실행법</h2>
1. manage.py이 있는 파일에 들어가서 VScode를 실행한다. <br>
2. '. myvenv/Scripts/activate'를 입력해 가상환경을 활성화 한다. <br>
3. 'python manage.py runserver'를 입력해 서버를 실행한다!
